## 📦 How to Use

1. Clone this repository:
   ```sh
   git clone https://github.com/joaootaviofm/responsive-navbar.git
   ```
2. Navigate to the project folder:
   ```sh
   cd responsive-navbar
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the project:
   ```sh
   npm run dev
   ```
